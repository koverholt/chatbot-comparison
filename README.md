# bot-comparison

## Build and deploy the app

1. Clone this repository and cd to the `site/` directory.
1. Install dependencies with NPM

   ```
   npm install
   ```

1. Build the site

   ```
   npm run build
   ```

1. Deploy the app

   ```
   firebase deploy
   ```

## Access the app

[https://chatbot-comparison.web.app](https://chatbot-comparison.web.app)
